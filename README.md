# SwipeMenuJS

### Usage:

HTML:
```html
<head>
    <link href="https://superzombi.github.io/SwipeMenuJS/swipe_menu.css" rel="stylesheet">
    <script src="https://superzombi.github.io/SwipeMenuJS/swipe_menu.js"></script>
</head>
```

JavaScript:
```javascript
var myMenu = document.getElementById("myMenu")
swipeMenu(myMenu)
```
